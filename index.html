How to Use TripShare
➕

Adding an Expense

Tap + to add an expense. Select category, enter amount, choose who paid, pick payment method (Cash / UPI / Credit / Debit / Net Banking), and how to split.

🔗

Sharing Your Trip

Tap Share on the top bar to get the invite code. Your trip-mates enter this code to join — all expenses sync in real-time when Firebase is configured.

👥

Managing Members

Add all people on the trip. Each gets a colour-coded avatar. Members are used to track who paid and who owes.

💸

Settle Up

The app calculates the minimum number of payments needed. See Settle Up on the dashboard or the full plan in Stats.

📶

Offline Mode

Add expenses without internet. They sync automatically once you reconnect — nothing is lost.

⬅️

Swipe to Delete

Swipe left on any expense item to reveal the delete action.

🔐

Biometric Lock

The app uses your device's biometric / screen lock to protect access each time you open it.

🌍

Currency & Localization

Change currency in Settings. Supports INR, USD, EUR, GBP, AED, SGD and more. Numbers are formatted per locale.

🔥 Enable Real-time Sync
For multiple users to see expenses in real-time, set up Firebase Firestore (free — no account/login needed):
1. Go to console.firebase.google.com
2. Click Add Project → name it (e.g. tripshare) → Create
3. Left sidebar → Build → Firestore Database
4. Click Create database → choose Test mode → Done
5. Left sidebar → ⚙️ Project Settings → General tab
6. Scroll to Your apps → click </> Web icon → Register
7. Copy the firebaseConfig object shown
8. Open index.html → find FIREBASE_CONFIG → paste your values ✅
💡 Tip: Use npx serve . or npx http-server . in the project folder to serve locally, or deploy to Netlify/Vercel for internet access.
